## Getting Started

All commands designed to run from the project root.

### Build

Start with building to make @elex/content and @elex/components modules available.

```
pnpm install
pnpm run build
```

### Develop

Run a dev server that responds to changes in all packages.

```
pnpm run dev
```

### Preview

Preview the build.

```
pnpm run preview
```
